### Usage:
1. clone the repo

2. install requirements
    ```bash
    pip install -r requirements.txt
    ```
3. run game.py

4. Controls
    W: Move up
    A: Move left
    S: Move down
    D: Move right
    M: Toggle minimap
    Q: Quit the game
